# Synent Task 4 – Interactive Sales Dashboard using Power BI

## 📌 Project Overview

This project was developed as part of the Synent Technologies Data Science Internship Program.

The objective of this task is to transform raw CSV data into an interactive business dashboard that helps users analyze sales performance, profit trends, regional performance, and product-level insights. The dashboard enables decision-makers to explore data dynamically through filters and visualizations.

---

## 🎯 Problem Statement

Organizations generate large volumes of sales data every day. Analyzing this data manually is difficult and time-consuming.

The goal of this project is to build an interactive dashboard that converts raw sales data into meaningful visual insights, helping stakeholders monitor business performance and make data-driven decisions.

## 🔄 Project Workflow

### 1. Data Collection

Imported the sales dataset in CSV format into Power BI.

### 2. Data Cleaning

Performed the following preprocessing steps:

* Checked for missing values
* Removed duplicate records
* Verified data types
* Formatted date columns
* Validated numerical fields

### 3. Data Modeling

Created measures and calculated fields for:

* Total Sales
* Total Profit
* Total Orders
* Profit Margin

### 4. Dashboard Development

Designed an interactive dashboard with:

* KPI Cards
* Sales Trend Analysis
* Profit Analysis
* Category Performance
* Regional Performance
* Top Products Analysis
* Dynamic Filters
